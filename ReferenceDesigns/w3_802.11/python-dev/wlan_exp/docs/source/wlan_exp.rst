Framework
*********

The 802.11 Reference Design Experiments framework communicates with WARP nodes 
in order to control their behavior and gather status / log information.  

Examples of wlan_exp experiments can be found at 
http://warpproject.org/trac/wiki/802.11/wlan_exp/examples
